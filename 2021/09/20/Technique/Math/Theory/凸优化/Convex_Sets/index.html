Where 1\boldsymbol{1}1 denotes the vector with all entries one. The affine dimension of this simplex is kkk, so it is sometimes referred to as a kkk-dimensional simplex in Rn\boldsymbol{R}^nRn.

8. Positive semidefinite cone

\qquadWe use the notation Sn\boldsymbol{S}^nSn to denote the set of symmetric n×nn \times nn×n matrices,

Sn={X∈Rn×n∣X=XT}(8.1)\begin{array}{c} \boldsymbol{S}^n = \{X \in \boldsymbol{R}^{n \times n} | X = X^T\} \tag{8.1} \end{array} Sn={X∈Rn×n∣X=XT}​(8.1)

which is a vector space with dimension n(n+1)2\frac{n(n+1)}{2}2n(n+1)​. We use the notation S+n\boldsymbol{S}_+^nS+n​ to denote the set of symmetric positive semidefinite matrices:

S+n={X∈Sn∣X⪰0}(8.2)\begin{array}{c} \boldsymbol{S}_+^n = \{X \in \boldsymbol{S}^n | X \succeq 0\} \tag{8.2} \end{array} S+n​={X∈Sn∣X⪰0}​(8.2)

and the notation S++n\boldsymbol{S}_{++}^nS++n​ to denote the set of symmetric positive definite matrices:

S++n={X∈Sn∣X⪰0}(8.3)\begin{array}{c} \boldsymbol{S}_{++}^n = \{X \in \boldsymbol{S}^n | X \succeq 0\} \tag{8.3} \end{array} S++n​={X∈Sn∣X⪰0}​(8.3)

(This notation is meant to be analogous to R+\boldsymbol{R}_+R+​, which denotes the nonnegative reals, and R++\boldsymbol{R}_{++}R++​, which denotes the positive reals.)

\qquadThe set S+n\boldsymbol{S}_+^nS+n​ is a positive semidefinite cone. Obviously, S+n\boldsymbol{S}_+^nS+n​ is also a convex cone: if θ1,θ2≥0\theta_1,\theta_2 \geq 0θ1​,θ2​≥0 and A,B∈S+nA,B \in \boldsymbol{S}_+^nA,B∈S+n​, then θ1A+θ2B∈S+n\theta_1 A + \theta_2 B \in \boldsymbol{S}_+^nθ1​A+θ2​B∈S+n​. This can be seen directly from the definition of positive semidefiniteness: for any x∈Rnx \in \boldsymbol{R}^nx∈Rn, we have

xT(θ1A+θ2B)x=θ1xTAx+θ2xTBx≥0(8.4)\begin{array}{c} x^T(\theta_1 A + \theta_2 B)x = \theta_1 x^T A x + \theta_2 x^T B x \geq 0 \tag{8.4} \end{array} xT(θ1​A+θ2​B)x=θ1​xTAx+θ2​xTBx≥0​(8.4)

if A⪰0,B⪰0A \succeq 0, B \succeq 0A⪰0,B⪰0 and θ1,θ2≥0\theta_1, \theta_2 \geq 0θ1​,θ2​≥0.
