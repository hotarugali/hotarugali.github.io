1. 随机事件

1.1 信息量与不确定度

事件发生的概率越大，它发生后提供的信息量就越小；事件发生的概率越小，一旦该事件发生，它发生后提供的信息量就越大。因此，事件 xxx 的信息量 I(x)I(x)I(x) 应该是该事件概率的函数

I(x)=f[p(x)](1.1)\begin{align*} I(x) = f[p(x)] \tag{1.1} \end{align*} I(x)=f[p(x)]​(1.1)

函数 I(x)I(x)I(x) 应满足的四大性质：

  1. f[p(x)]f[p(x)]f[p(x)] 应该是 p(x)p(x)p(x) 的单调递减函数；
  2. 当 p(x)=1p(x) = 1p(x)=1 时，f[p(x)]=0f[p(x)] = 0f[p(x)]=0；
  3. 当 p(x)=0p(x) = 0p(x)=0 时，f[p(x)]=∞f[p(x)] = \inftyf[p(x)]=∞；
  4. 两个独立事件的联合信息量应该等于各自信息量之和。

1.2 自信息量

  • 定义：任何随机事件的自信息量定义为该事件发生概率的对数的负值。假设事件 xxx 发生的概率为 p(x)p(x)p(x)，则其自信息定义为

    I(x)=−log⁡2p(x)(1.2)\begin{align*} I(x) = -\log_{2} p(x) \tag{1.2} \end{align*} I(x)=−log2​p(x)​(1.2)

    自信息量衡量的是随机事件的不确定性。事件的不确定性越大，其自信息量也越大，反之亦然。

  • 性质：式 (1.2)(1.2)(1.2) 中定义的自信息满足式 (1.1)(1.1)(1.1) 中 f[p(x)]f[p(x)]f[p(x)] 应当满足的四大性质，即

    1. 函数 I(x)I(x)I(x) 是 p(x)p(x)p(x) 的递减函数；

    2. 当 p(x)=1p(x) = 1p(x)=1 时，I(x)=0I(x) = 0I(x)=0；

    3. 当 p(x)=0p(x) = 0p(x)=0 时，I(x)=∞I(x) = \inftyI(x)=∞；

    4. 两个独立事件的联合信息量等于各自信息量之和

    I(x1x2)=−log⁡2p(x1x2)=−log⁡2p(x1)p(x2)=I(x1)+I(x2)\begin{array}{c} I(x_1x_2) = -\log_{2} p(x_1x_2) = -\log_{2} p(x_1)p(x_2) = I(x_1) + I(x_2) \end{array} I(x1​x2​)=−log2​p(x1​x2​)=−log2​p(x1​)p(x2​)=I(x1​)+I(x2​)​

1.3 联合自信息量

  • 定义：二维联合集 XYXYXY 上的元素 (xy)(x y)(xy) 的联合自信息量定义为

    I(xy)=−log⁡2p(xy)(1.3)\begin{align*} I(x y) = -\log_{2} p(x y) \tag{1.3} \end{align*} I(xy)=−log2​p(xy)​(1.3)

    联合自信息量衡量的是多个事件同时出现的不确定性。

1.4 条件自信息量

  • 定义：事件 xxx 在事件 yyy 给定的条件下的条件自信息定义为

    I(x∣y)=−log⁡2p(x∣y)(1.4)\begin{align*} I(x|y) = -\log_{2} p(x|y) \tag{1.4} \end{align*} I(x∣y)=−log2​p(x∣y)​(1.4)

    条件自信息量衡量的是已知事件 yyy 之后仍然保留的关于 xxx 的不确定性。

1.5 互信息量

  • 定义：随机事件 yyy 的出现给出关于事件 xxx 的信息量定义为二者的互信息量，定义为

    I(x;y)=log⁡2p(x∣y)p(x)=log⁡2p(xy)p(x)p(y)(1.5)\begin{align*} I(x; y) = \log_{2} \frac{p(x|y)}{p(x)} = \log_2 \frac{p(xy)}{p(x) p(y)} \tag{1.5} \end{align*} I(x;y)=log2​p(x)p(x∣y)​=log2​p(x)p(y)p(xy)​​(1.5)

    本身的不确定性，减去知道事件 yyy 之后仍然保留的不确定性，即由 yyy 所提供的关于 xxx 的信息量或由 yyy 所消除的关于 xxx 的不确定性，称为事件 xxx 与 yyy 的互信息量。

  • 性质：

    1. I(x;y)=I(x)−I(x∣y)I(x;y) = I(x) - I(x|y)I(x;y)=I(x)−I(x∣y)。

      证明：I(x;y)=log⁡2p(x∣y)p(x)=log⁡21p(x)−log⁡21p(x∣y)=I(x)−I(x∣y)I(x;y) = \log_{2}{p(x|y) \over p(x)} = \log_{2}{1 \over p(x)} - \log_{2}{1 \over p(x|y)} = I(x) - I(x|y)I(x;y)=log2​p(x)p(x∣y)​=log2​p(x)1​−log2​p(x∣y)1​=I(x)−I(x∣y)
      即互信息量 = 原有不确定性 - 尚存在的不确定性

    2. 互易性：由 yyy 所提供的关于 xxx 的信息量 = 由 xxx 所提供的关于 yyy 的信息量，即 I(x;y)=I(y;x)I(x;y) = I(y;x)I(x;y)=I(y;x)。

      证明：I(x;y)=log⁡2p(x∣y)p(x)=log⁡2p(x∣y)p(y)p(x)p(y)=log⁡2p(xy)p(x)p(y)=log⁡2p(y∣x)p(y)=I(y;x)I(x;y) = \log_{2}{p(x|y) \over p(x)} = \log_{2}{p(x|y)p(y) \over p(x)p(y)} = \log_{2}{p(xy) \over p(x)p(y)} = \log_{2}{p(y|x) \over p(y)} = I(y;x)I(x;y)=log2​p(x)p(x∣y)​=log2​p(x)p(y)p(x∣y)p(y)​=log2​p(x)p(y)p(xy)​=log2​p(y)p(y∣x)​=I(y;x)

    3. 当事件 xxx、yyy 统计独立时，互信息量为 0；即两个事件相互独立时，一个事件不能提供另一个事件的任何信息 。

      证明：I(x;y)=log⁡2p(x∣y)p(x)=log⁡2p(x)p(x)=0I(x;y) = \log_{2}{p(x|y) \over p(x)} = \log_{2}{p(x) \over p(x)} = 0I(x;y)=log2​p(x)p(x∣y)​=log2​p(x)p(x)​=0

    4. 互信息量可正可负。正表示 yyy 的出现有利于确定 xxx 的发生；负表示 yyy 的出现不利于确定 xxx 的发生 。

      无论正负，互信息量的绝对值越大，xxx 和 yyy 的关系越密切。

    5. 互信息量不大于任一事件的自信息量 。

      证明：I(x;y)=I(y;x)I(x;y) = I(y;x)I(x;y)=I(y;x)
            I(x;y)=log⁡2p(x∣y)p(x)≤log⁡21p(x)=I(x)I(x;y) = \log_{2}{p(x|y) \over p(x)} \leq \log_{2}{1 \over p(x)} = I(x)I(x;y)=log2​p(x)p(x∣y)​≤log2​p(x)1​=I(x)
            I(y;x)=log⁡2p(y∣x)p(y)≤log⁡21p(y)=I(y)I(y;x) = \log_{2}{p(y|x) \over p(y)} \leq \log_{2}{1 \over p(y)} = I(y)I(y;x)=log2​p(y)p(y∣x)​≤log2​p(y)1​=I(y)

  • 单位及换算：同自信息量。

1.6 条件互信息量

  • 定义：联合事件集 XYZXYZXYZ 中，在给定 zzz 的条件下，xxx 与 yyy 之间的互信息量称为条件互信息量，定义为：

    I(x;y∣z)=log⁡2p(x∣yz)p(x∣z)(1.6)\begin{align*} I(x;y|z) = \log_{2}{p(x|yz) \over p(x|z)} \tag{1.6} \end{align*} I(x;y∣z)=log2​p(x∣z)p(x∣yz)​​(1.6)

    条件互信息量衡量的是已知了 zzz 后，yyy 提供关于 xxx 的信息量

  • 性质：I(x;y∣z)=I(x;yz)−I(x;z)I(x;y|z) = I(x;yz) - I(x;z)I(x;y∣z)=I(x;yz)−I(x;z)。

    证明：I(x;y∣z)=log⁡2p(x∣yz)p(x∣z)=log⁡2p(x∣yz)p(x)p(x∣z)p(x)=log⁡2p(x∣yz)p(x)−log⁡2p(x∣z)p(x)=I(x;yz)−I(x;z)I(x;y|z) = \log_{2}{p(x|yz) \over p(x|z)} = \log_{2}{p(x|yz)p(x) \over p(x|z)p(x)} = \log_{2}{p(x|yz) \over p(x)} - \log_{2}{p(x|z) \over p(x)} = I(x;yz) - I(x;z)I(x;y∣z)=log2​p(x∣z)p(x∣yz)​=log2​p(x∣z)p(x)p(x∣yz)p(x)​=log2​p(x)p(x∣yz)​−log2​p(x)p(x∣z)​=I(x;yz)−I(x;z)

离散事件集 X=x1,x2,⋯ ,xqX = {x_1,x_2,\cdots,x_q}X=x1​,x2​,⋯,xq​，其概率分布表示为：

  • 定义：条件自信息量 I(x∣y)I(x|y)I(x∣y) 的概率均值称为条件熵，定义为：

2.4 互信息

  • 定义：离散事件集合 XXX 包含离散事件 YYY 的平均信息量称为平均互信息量，简称互信息，定义为

    I(X;Y)=EXY[I(x;y)]=−∑XYp(xy)log⁡2p(x∣y)p(x)=−∑XYp(xy)log⁡2p(xy)p(x)p(y)(2.4)\begin{align*} I(X;Y) = E_{XY}[I(x; y)] = -\sum_{XY} p(x y) \log_{2} \frac{p(x|y)}{p(x)} = -\sum_{XY} p(x y) \log_2 \frac{p(xy)}{p(x) p(y)} \tag{2.4} \end{align*} I(X;Y)=EXY​[I(x;y)]=−XY∑​p(xy)log2​p(x)p(x∣y)​=−XY∑​p(xy)log2​p(x)p(y)p(xy)​​(2.4)
