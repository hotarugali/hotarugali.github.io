P4462「[CQOI2018]异或序列」

发表于2021-08-27|更新于2023-12-17|TechniqueACM题解
|字数总计:1.6k|阅读时长:7分钟|阅读量:

1. 题目

题目链接：P4462「[CQOI2018]异或序列」 。

题目描述

已知一个长度为n的整数数列 a1,a2,⋯ ,ana_1,a_2,\cdots,a_na1​,a2​,⋯,an​，给定查询参数 l,rl,rl,r，问在 al,al+1,⋯ ,ara_l,a_{l+1},\cdots,a_ral​,al+1​,⋯,ar​ 区间内，有多少子序列满足异或和等于 kkk。也就是说，对于所有的 x,yx,yx,y（l≤x≤y≤rl \leq x \leq y \leq rl≤x≤y≤r），能够满足 ax⨁ax+1⨁⋯⨁ay=ka_x \bigoplus a_{x+1} \bigoplus \cdots \bigoplus a_y = kax​⨁ax+1​⨁⋯⨁ay​=k 的 x,yx,yx,y 有多少组。

输入格式

输入文件第一行，为 333 个整数 n,m,kn,m,kn,m,k。

第二行为空格分开的 nnn 个整数，即 a1,a2,..ana_1,a_2,..a_na1​,a2​,..an​。

接下来 mmm 行，每行两个整数 lj,rjl_j,r_jlj​,rj​，表示一次查询。

输出格式

输出文件共 mmm 行，对应每个查询的计算结果。

输入输出样例

输入 #1

输出 #1

说明/提示

对于 30%30\%30% 的数据，1≤n,m≤10001 \leq n, m \leq 10001≤n,m≤1000。

对于 100%100\%100% 的数据，1≤n,m≤105,0≤k,ai≤105,1≤lj≤rj≤n1 \leq n, m \leq 10^5, 0 \leq k, a_i \leq 10^5,1 \leq l_j \leq r_j \leq n1≤n,m≤105,0≤k,ai​≤105,1≤lj​≤rj​≤n。

2. 题解

分析

  • 首先需要了解异或运算的优美性质。
  1. 设 si=a1⨁a2⨁⋯⨁ais_i = a_1 \bigoplus a_2 \bigoplus \cdots \bigoplus a_isi​=a1​⨁a2​⨁⋯⨁ai​，则有

al⨁al+1⨁⋯⨁ar=sl−1⨁sr\begin{array}{c} a_l \bigoplus a_{l+1} \bigoplus \cdots \bigoplus a_r = s_{l-1} \bigoplus s_r \end{array} al​⨁al+1​⨁⋯⨁ar​=sl−1​⨁sr​​

  1. 假设 a⨁b=ca \bigoplus b = ca⨁b=c，则有

a=b⨁cb=a⨁c\begin{array}{c} a = b \bigoplus c \\ b = a \bigoplus c \\ \end{array} a=b⨁cb=a⨁c​

因此，对于原题，我们可以将其转换为：在区间 [l,r][l,r][l,r] 内，有多少对 sx−1,sys_{x-1},s_ysx−1​,sy​（l≤x≤y≤rl \leq x \leq y \leq rl≤x≤y≤r）满足 sx−1⨁sy=ks_{x-1} \bigoplus s_y = ksx−1​⨁sy​=k，其中，si=a1⨁⋯⨁ai−1s_i = a_1 \bigoplus \cdots \bigoplus a_{i-1}si​=a1​⨁⋯⨁ai−1​，即 sss 数组为 aaa 数组的异或前缀和。

  • 然后计算一下时间复杂度，O(mn)O(m\sqrt{n})O(mn​) 可以过，而且可以离线查询，因此可以考虑用莫队算法。接下的问题就是如何实现相邻查询区间的 O(1)O(1)O(1) 转移。

假设序列 sss 的长度为 nnn，若已知 [l,r][l,r][l,r] 区间的答案为 ansl,rans_{l,r}ansl,r​，且 countcountcount 数组记录了数组 sss 在 [l,r][l,r][l,r] 区间中各个数的出现次数，则区间 [l,r+1][l,r+1][l,r+1]（其它三个相邻区间的分析类似）的答案为

ansl,r+1=ansl,r+count[s[r+1]k]\begin{array}{c} ans_{l,r+1} = ans_{l,r} + count[s[r+1]^k] \end{array} ansl,r+1​=ansl,r​+count[s[r+1]k]​

因此，只要在区间改变时维护好 countcountcount 数组，就可实现相邻查询区间 O(1)O(1)O(1) 复杂度的转移。

注意

需要判断一下应该开的数组大小以及数据类型是否会溢出：

  • 数组大小：由于 0≤k,ai≤1050 \leq k, a_i \leq 10^50≤k,ai​≤105，因此它们之间的异或结果最大为 2⌊log⁡2105⌋+1−12^{\lfloor\log_2{10^5}\rfloor + 1} - 12⌊log2​105⌋+1−1，即异或的结果会超出 10510^5105，但始终能用 ⌊log⁡2105⌋\lfloor\log_2{10^5}\rfloor⌊log2​105⌋ 位二进制表示。而 ⌊log⁡2105⌋\lfloor\log_2{10^5}\rfloor⌊log2​105⌋ 位二进制所能表示的最大数为 2⌊log⁡2105⌋+1−12^{\lfloor\log_2{10^5}\rfloor + 1} - 12⌊log2​105⌋+1−1，又 2⌊log⁡2105⌋+1−1≤2×105−12^{\lfloor\log_2{10^5}\rfloor + 1} - 1 \leq 2 \times 10^5 - 12⌊log2​105⌋+1−1≤2×105−1，因此数组长度需要开到 2×1052 \times 10^52×105 才合适。
  • 数据类型：由于区间 [l,r][l,r][l,r] 中最多有 Cr−l+12C_{r-l+1}^2Cr−l+12​ 种两两配对，且 r−l+1≤105r-l+1 \leq 10^5r−l+1≤105，因此 Cr−l+12<=5×104×(105−1)≈5×109C_{r-l+1}^2 <= 5 \times 10^4 \times (10^5 - 1) \approx 5 \times 10^9Cr−l+12​<=5×104×(105−1)≈5×109，所以 int 数据类型其实是不够用的，要用 long long。

虽然这道没有卡这两点，但不可忽略它们。另一道基本相同的题 CF617E XOR and Favorite Number 就考虑了这两点。
