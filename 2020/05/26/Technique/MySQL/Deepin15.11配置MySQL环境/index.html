1. 添加 APT Repository

从官网上下载对应的 MySQL APT Repository 配置包，然后安装该 mysql-apt-config_x.y.z_all.deb 包（具体取决于所下载的 deb 包）。

在安装该 APT Repository 配置包过程中，会要求进行 mysql-apt-config 设定，包括 mysql 版本的选择以及 mysql 的一些工具包软件的安装选择，具体选择取决个个人。
