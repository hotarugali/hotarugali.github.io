Rime中文输入方案

发表于2020-10-25|更新于2021-04-11|InputMethod
|字数总计:376|阅读时长:1分钟|阅读量:

【注】更多功能参见 Rime 官网。

1. 简介

Rime 可谓是一款「神级」中文输入方案，集多种输入法于一身，诸如拼音、注音、仓颉、速成、五笔、双拼、宫保拼音、粤拼、吴语、中古汉语拼音、五笔画、Emoji、国际音标……
同时 Rime 也是一款开源跨平台的软件，支持平台有：

  • 【中州韻】 ibus-rime、fcitx-rime →\rightarrow→ Linux
  • 【小狼毫】 Weasel →\rightarrow→ Windows
  • 【鼠鬚管】 Squirrel →\rightarrow→ Mac OS X

2. 安装

各个平台上的具体安装以及更多的输入方案的安装见官方地址。

3. 选择输入方案

安装完成后，按快捷键 Ctrl + ` 或 F4 唤出输入方案选单，然后根据个人需求切换输入方案。

4. 附录

Rime 还允许用户 DIY 自己的输入方案，具体方法见 Rime 定制指南 和 Rime 输入方案设计书。

5. 同步数据

Rime 还提供保存用户输入数据的功能，保存的用户输入数据用于 Rime 提供更加个性化的输入方案，包括用户常用的字、词等；同时，也可以将保存的用户输入数据移植到其他新平台等。设置 Rime 用户数据同步保存文件夹：
