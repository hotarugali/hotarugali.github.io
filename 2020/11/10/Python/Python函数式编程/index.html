1. иЈ…йҘ°еҷЁпјҲdecoratorпјү

дёҠйқўзҡ„ log е°ұжҳҜдёҖдёӘ decoratorгҖӮPython зҡ„ @ иҜӯжі•еҸҜд»ҘжҠҠ decorator зҪ®дәҺеҮҪж•°зҡ„е®ҡд№үеӨ„пјҡ

жӯӨж—¶и°ғз”Ё now еҮҪж•°пјҢдјҡеңЁиҝҗиЎҢ now еүҚиҝҗиЎҢ log(now) жү“еҚ°ж—Ҙеҝ—гҖӮ
