2. 矩阵

2.1 定义

设 A∈Cn×n A \in \mathbb{C}^{n \times n} A∈Cn×n：

  • 若 A∗A=AA∗ A^* A = A A^* A∗A=AA∗，则称 AAA 为正规矩阵。

  • 若 A=A∗ A = A^* A=A∗，则称 AAA 为 Hermite 矩阵；若 A=−A∗ A = -A^* A=−A∗，则称 AAA 为反 Hermite 矩阵。

  • 若 A∗A=I A^* A = I A∗A=I，则称 AAA 为酋矩阵。

2.2 性质

  • Hermite 矩阵，反 Hermite 矩阵，酋矩阵都是正规矩阵。
  • 实 Hermite 矩阵就是实对称矩阵，实酋矩阵就是实正交矩阵。

一个方阵的全体特征值的集合称为该方阵的谱。

3. 行列式

3.1 定义

方阵的行列式记作 det(A)\mathrm{det}(A)det(A) 或 ∣A∣|A|∣A∣，其定义如下：

det(A)=∑σ∈Sn(sgn(σ)∏i=1nai,σi)\mathrm{det}(A) = \sum_{\sigma \in S_n} \left( \mathrm{sgn}(\sigma) \prod_{i=1}^n a_{i,\sigma_i} \right) det(A)=σ∈Sn​∑​(sgn(σ)i=1∏n​ai,σi​​)

其中，SnS_nSn​ 是元素 1∼n1 \sim n1∼n 的置换群，sgn\mathrm{sgn}sgn 为 sign 函数。

置换 σ\sigmaσ 的 sign 函数只返回两个值：+1+1+1 如果置换 σ\sigmaσ 是偶的，−1-1−1 如果置换 σ\sigmaσ 是奇的。

逆序数为奇数的置换叫做奇置换，逆序数为偶数的置换叫做偶置换。

3.2 性质

3.2.1 基本性质

  • det(cA)=cndet(A)\mathrm{det}(cA) = c^n \mathrm{det}(A)det(cA)=cndet(A)

  • det(A⊤)=det(A)\mathrm{det}(A^\top) = \mathrm{det}(A)det(A⊤)=det(A)

  • 若 A,BA, BA,B 均为方阵，则有：det(AB)=det(A)det(B)\mathrm{det}(AB) = \mathrm{det}(A) \mathrm{det}(B)det(AB)=det(A)det(B)

  • 若方阵 AAA 可逆，则有：det(A−1)=(det(A))−1\mathrm{det}(A^{-1}) = (\mathrm{det}(A))^{-1}det(A−1)=(det(A))−1

  • det(A)=∑i=1n(−1)i+jaijMij\mathrm{det}(A) = \sum_{i=1}^n (-1)^{i+j} a_{ij} M_{ij}det(A)=∑i=1n​(−1)i+jaij​Mij​，其中 aja_jaj​ 是选择展开的列，MijM_{ij}Mij​ 是对应的余子式，一般记 Cij=(−1)i+jMijC_{ij} = (-1)^{i+j} M_{ij}Cij​=(−1)i+jMij​ 为代数余子式，则有：det(A)=∑i=1naijCij\mathrm{det}(A) = \sum_{i=1}^n a_{ij} C_{ij}det(A)=∑i=1n​aij​Cij​

记 AAA 的伴随矩阵为 adj(A)\mathrm{adj}(A)adj(A)，其定义为：(adj(A))ij=(−1)i+jMji(\mathrm{adj}(A))_{ij} = (-1)^{i+j} M_{ji}(adj(A))ij​=(−1)i+jMji​。

  • 若方阵 AAA 可逆，则有：A−1=1det(A)adj(A)A^{-1} = \frac{1}{\mathrm{det}(A)} \mathrm{adj}(A)A−1=det(A)1​adj(A)

  • det(A)=∏i=1nλi\mathrm{det}(A) = \prod_{i=1}^n \lambda_idet(A)=∏i=1n​λi​，其中所有的 λi\lambda_iλi​ 为方阵 AAA 的特征值

3.2.2 舒尔补

若方阵 X∈SnX \in \mathbb{S}^nX∈Sn，将其使用分块矩阵表达如下：

X=[ABB⊤C]X = \left[ \begin{matrix} A & B \\ B^\top & C \end{matrix} \right] X=[AB⊤​BC​]

其中，A∈SkA \in \mathbb{S}^kA∈Sk。若 det(A)≠0\mathrm{det}(A) \neq 0det(A)=0，则称矩阵

S=C−B⊤A−1BS = C - B^\top A^{-1} B S=C−B⊤A−1B

为 AAA 在 XXX 中的舒尔补。舒尔补具有如下性质：

  • det(X)=det(A)det(S)\mathrm{det}(X) = \mathrm{det}(A) \mathrm{det}(S)det(X)=det(A)det(S)
  • XXX 正定当且仅当 AAA 正定且 SSS 正定
  • 若 AAA 正定，则 XXX 半正定当且仅当 SSS 半正定。

4. 迹

4.1 定义

方阵 AAA 的迹记作 tr⁡(A)\operatorname{tr}(A)tr(A)，其定义如下：

tr⁡(A)=∑iAii\operatorname{tr}(A) = \sum_{i} A_{ii} tr(A)=i∑​Aii​

4.2 性质

4.2.1 基本性质

  • tr⁡(cA)=ctr⁡(A)\operatorname{tr}(cA) = c\operatorname{tr}(A)tr(cA)=ctr(A)
  • tr⁡(A+B)=tr⁡(A)+tr⁡(B)\operatorname{tr}(A + B) = \operatorname{tr}(A) + \operatorname{tr}(B)tr(A+B)=tr(A)+tr(B)
  • tr⁡(A)=tr⁡(A⊤)\operatorname{tr}(A) = \operatorname{tr}(A^\top)tr(A)=tr(A⊤)

设 A,B∈Rm×nA, B \in \mathbb{R}^{m \times n}A,B∈Rm×n，则有：

tr(A⊤B)=tr(AB⊤)=tr(B⊤A)=tr(BA⊤)=∑im∑jnaijbij\mathrm{tr}(A^\top B) = \mathrm{tr}(AB^\top) = \mathrm{tr}(B^\top A) = \mathrm{tr}(BA^\top) = \sum_{i}^m \sum_{j}^n a_{ij} b_{ij} tr(A⊤B)=tr(AB⊤)=tr(B⊤A)=tr(BA⊤)=i∑m​j∑n​aij​bij​

若 m=nm = nm=n，即 AAA 和 BBB 均为方阵，则进一步有：

tr(A⊤B)=tr(AB⊤)=tr(B⊤A)=tr(BA⊤)=∑im∑jnaijbij=∑ij(A⊙B)ij\mathrm{tr}(A^\top B) = \mathrm{tr}(AB^\top) = \mathrm{tr}(B^\top A) = \mathrm{tr}(BA^\top) = \sum_{i}^m \sum_{j}^n a_{ij} b_{ij} = \sum_{ij} (A \odot B)_{ij} tr(A⊤B)=tr(AB⊤)=tr(B⊤A)=tr(BA⊤)=i∑m​j∑n​aij​bij​=ij∑​(A⊙B)ij​
