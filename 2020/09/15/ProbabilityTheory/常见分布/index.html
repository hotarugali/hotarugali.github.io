1. 离散分布

设 q=1−pq = 1 - pq=1−p ：

1.1 0-1 分布（伯努利分布）

P(X=1)=pP(X=0)=q{\begin{array}{c} P(X = 1) = p \\ P(X = 0) = q \end{array}}P(X=1)=pP(X=0)=q​

1.2 二项分布（n 重伯努利分布）

P(X=k)=Cnkpkqn−k{\begin{array}{c} P(X = k) = C_n^k p^k q^{n-k} \end{array}}P(X=k)=Cnk​pkqn−k​

记作 X∼B(n,p) X \sim B(n,p) X∼B(n,p) 。

1.3 多项式分布

多项式分布是二项分布的推广：

P(X=x1,⋯ ,xk)=n!x1!⋯xk!p1x1⋯pkxkpi≥0,  ∑pi=1xi∈{0,⋯ ,n},  ∑xi=n{\begin{array}{c} P(X = {x_1,\cdots,x_k}) = {n! \over x_1! \cdots x_k!}p_1^{x_1} \cdots p_k^{x_k} \\ p_i \geq 0, \ \ \sum p_i = 1 \\ x_i \in \{0,\cdots,n\}, \ \ \sum x_i = n \end{array}}P(X=x1​,⋯,xk​)=x1​!⋯xk​!n!​p1x1​​⋯pkxk​​pi​≥0,  ∑pi​=1xi​∈{0,⋯,n},  ∑xi​=n​

1.4 泊松分布

P(X=k)=λkk!eλ, λ>0{\begin{array}{c} P(X = k) = {\lambda^k \over k!}e^\lambda, \ \lambda > 0 \end{array}}P(X=k)=k!λk​eλ, λ>0​

记作 X∼π(λ) X \sim \pi(\lambda) X∼π(λ) 。

1.5 超几何分布

P(X=k)=CMk⋅CN−Mn−kCNn{\begin{array}{c} P(X = k) = {C_M^k \cdot C_{N-M}^{n-k} \over C_N^n} \end{array}}P(X=k)=CNn​CMk​⋅CN−Mn−k​​​

记作 X∼H(n,N,M) X \sim H(n,N,M) X∼H(n,N,M) 。

1.6 几何分布

P(X=k)=qk−1p{\begin{array}{c} P(X = k) = q^{k-1}p \end{array}}P(X=k)=qk−1p​

记作 X∼G(p) X \sim G(p) X∼G(p) 。
