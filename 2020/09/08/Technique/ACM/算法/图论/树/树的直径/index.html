树的直径

发表于2020-09-08|更新于2023-12-17|TechniqueACM算法图论树
|字数总计:1.5k|阅读时长:6分钟|阅读量:

1. 简介

树中所有简单路径的最大值即为树的直径，可以通过两次 DFS 或者树形 DP 在 O(n)O(n)O(n) 时间内求解。

2. 思路

2.1 两次 DFS

  • 首先将任意一个结点 uuu 看作是树根，然后进行 DFS 求出最远的结点 sss；则 sss 一定是树的直径的其中一个端点。

证明

  假设结点 s′s's′ 和 t′t't′ 之间唯一的简单路径为树的直径，(a,b)(a,b)(a,b) 表示结点 aaa 和 bbb 之间的唯一的简单路径的长度。若该最远点 sss 不是树的直径的一个端点，则对于当前根结点 uuu 来说：

  1. 如果 s′s's′ 或 t′t't′ 二者有一个不与 sss 在同一个子树（假设为 t′t't′），则由于

(s′,t′)≤(s′,u)+(u,t′)≤(s,u)+(u,t′)=(s,t′)\begin{array}{c} (s',t') \leq (s',u)+(u,t') \leq (s,u)+(u,t') = (s,t') \end{array} (s′,t′)≤(s′,u)+(u,t′)≤(s,u)+(u,t′)=(s,t′)​

与 (s′,t′)(s',t')(s′,t′) 为直径而 sss 不是直径的端点矛盾。

  1. 如果 s′s's′、t′t't′ 均与 sss 在同一个子树，易知树的直径 (s′,t′)(s',t')(s′,t′) 没有经过树根 uuu。设该子树根为 u′u'u′，则易知 u′≠s′u' \ne s'u′=s′ 且 u′≠t′u' \ne t'u′=t′ 且 u′≠su' \ne su′=s，于是 (u′,s)(u',s)(u′,s) 一定为子树 u′u'u′ 中的最大值，即 sss 为子树 u′u'u′ 的最远点，对子树 u′u'u′ 同样进行上述分析，以此类推。

综上可知，sss 一定是树的直径的一个端点。

推论：树中任意结点的最远点要么是 sss，要么是 ttt（(s,t)(s,t)(s,t) 为树的直径）。

2.2 树形 DP

  • 定义树的高度为树根到所有结点的简单路径的最大值，次高度为树根到所有结点的简单路径第二大的值（高度 ≥\geq≥ 次高度）。

  • 对于树中任意一个结点 vvv，若 vvv 到其子树 lil_ili​ 中的最远结点的距离为 LiL_iLi​，设所有 LiL_iLi​ 中最大的二者为 L1rt=hi+(li,v)L_{1rt} = h_i + (l_i,v)L1rt​=hi​+(li​,v) 和 L2nd=hj+(lj,v)L_{2nd} = h_j + (l_j,v)L2nd​=hj​+(lj​,v)，对应的 vvv 的子树为 lil_ili​ 和 ljl_jlj​（没有子树 = 子树对应的 LiL_iLi​ 为零，即没有足够的子树可以看作有 LiL_iLi​ 为零的对应子树），则子树 vvv 中过结点 vvv 的最长简单路径等于

L1rt+L2nd\begin{array}{c} L_{1rt} + L_{2nd} \end{array} L1rt​+L2nd​​

子树 vvv 的高度等于

max⁡{Lk=hk+lk}\begin{array}{c} \max\{L_k = h_k + l_k\} \end{array} max{Lk​=hk​+lk​}​

其中，kkk 取遍 vvv 的所有子树，L1rtL_{1rt}L1rt​ 和 L2ndL_{2nd}L2nd​ 即对应子树 vvv 的高度和次高度。

  • 由于树的直径一定是以树中某个结点 v′v'v′ 为根的子树中过结点 v′v'v′ 的最长简单路径，因此计算每个树结点 u′u'u′ 的过结点 u′u'u′ 的最长简单路径，同时维护一下最大值即可。

  • 计算每个树结点 u′u'u′ 的过结点 u′u'u′ 的最长简单路径就是一个树形 dp 问题：以当前结点为根的子树的高度和次高度的解取决于其子树的高度，而需要计算的最长简单路径即为 u′u'u′ 子树的高度和次高度之和。
